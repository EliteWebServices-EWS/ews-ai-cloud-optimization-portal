# DNS Migration

GitHub Pages DNS records were removed.

The production DNS configuration now consists of:

| Type | Host | Target |
|------|------|--------|
| ALIAS | @ | d287wcw6yqb2r1.cloudfront.net |
| CNAME | www | d287wcw6yqb2r1.cloudfront.net |

The ACM DNS validation CNAME records were retained for automatic certificate renewal.

A CAA record was added to authorize Amazon Certificate Manager:

```text
CAA
Host: @
Value: 0 issue "amazon.com"
```
