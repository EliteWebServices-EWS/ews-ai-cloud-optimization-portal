Welcome to Elite Web Services!

Hi Andrew,

We're excited to officially welcome you to the Elite Web Services (EWS) team.

Our goal is to build a modern AWS consulting practice while developing EWS-Lite, a cloud optimization portal that demonstrates cloud cost optimization, governance, FinOps, AI-assisted operations, and enterprise cloud best practices.

This project is a collaborative learning environment where everyone contributes, learns, and gains practical experience working on a real cloud consulting platform.
Project Mission

Our mission is to help organizations:

    Optimize AWS cloud costs

    Improve cloud security and governance

    Modernize cloud operations

    Implement FinOps best practices

    Use AI to improve operational efficiency

The website is our business storefront.

The EWS-Lite Portal is one of the services we provide.

Git Workflow

Our workflow is:

main
│
├── feature/your-feature
│
├── Commit
│
├── Push
│
├── Pull Request
│
└── Review
      │
      ▼
Mpho approves
      │
      ▼
Merge into main

Important Rules

    Never commit directly to main

    One feature per branch

    One Pull Request per feature

    Keep commits small and descriptive
